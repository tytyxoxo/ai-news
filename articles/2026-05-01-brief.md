## 1. Microsoft เปิดตัว Agent 365 และ Microsoft 365 E7 อย่างเป็นทางการ ราคา $99/ผู้ใช้/เดือน พร้อม AI หลายโมเดล

Microsoft เปิดตัว Agent 365 อย่างเป็นทางการในวันที่ 1 พฤษภาคม พร้อมกับแพ็กเกจ Microsoft 365 Enterprise 7 (E7) ในราคา 99 ดอลลาร์ต่อผู้ใช้ต่อเดือน ซึ่งรวม Microsoft 365 Copilot, Agent 365, Microsoft Entra Suite และระบบความปลอดภัยชั้นสูงไว้ในชุดเดียว Agent 365 ทำหน้าที่เป็น "ระบบควบคุมกลาง" สำหรับ AI agents ในองค์กร ช่วยให้ทีม IT สามารถตรวจสอบ กำกับดูแล และรักษาความปลอดภัยของ agents ทั้งหมดได้ในที่เดียว Wave 3 ของ Copilot รองรับ multi-model AI ทั้งจาก OpenAI และ Claude ของ Anthropic ฝังลึกใน Word, Excel, PowerPoint และ Outlook ทำให้สามารถทำงานซับซ้อนหลายขั้นตอนโดยอัตโนมัติ นับเป็นก้าวสำคัญในการนำ AI เข้าสู่งานองค์กรอย่างเป็นระบบและปลอดภัยในระดับ enterprise

## 2. SoftBank เตรียมสร้างบริษัทหุ่นยนต์ Roze AI สำหรับก่อสร้างดาต้าเซ็นเตอร์ ตั้งเป้า IPO มูลค่า $100 พันล้านดอลลาร์

SoftBank Group เปิดเผยแผนสร้างบริษัท AI และหุ่นยนต์ใหม่ชื่อ "Roze" โดย CEO Masayoshi Son เป็นผู้ขับเคลื่อน ความน่าสนใจคือ Roze ไม่ใช่บริษัท AI แบบทั่วไป แต่เป็นบริษัทที่ใช้หุ่นยนต์อัตโนมัติในการก่อสร้างดาต้าเซ็นเตอร์ ตั้งแต่การยกของหนัก การเชื่อม ไปจนถึงการประกอบอุปกรณ์ เพื่อลดเวลาและต้นทุนการก่อสร้าง SoftBank วางแผนนำ Roze เข้าตลาดหลักทรัพย์สหรัฐฯ ในช่วงปลายปี 2026 โดยตั้งเป้ามูลค่าถึง 100 พันล้านดอลลาร์ ถ้าสำเร็จจะเป็นสัญญาณที่ชัดเจนว่าโลกกำลังมองว่า physical AI สำหรับโครงสร้างพื้นฐาน AI นั้นมีมูลค่ามหาศาลพอๆ กับตัวโมเดล AI เอง และยังสะท้อนถึงความต้องการดาต้าเซ็นเตอร์ที่เติบโตอย่างรวดเร็วจนเกินกำลังแรงงานมนุษย์

## 3. GitHub ประกาศ "Availability First" หลัง AI agents ทำให้ระบบล่มซ้ำ โหลดเพิ่มขึ้น 30 เท่าจากที่คาดไว้

GitHub ออกแถลงการณ์ขอโทษผู้ใช้งานหลังระบบประสบปัญหาไม่เสถียรต่อเนื่องตลอดเดือนเมษายน โดย CTO Vlad Fedorov ยอมรับว่าสาเหตุหลักมาจาก AI agents ที่ใช้แพลตฟอร์มหนักกว่าที่คาดการณ์ไว้ถึง 30 เท่า ส่งผลให้ uptime เหลือเพียงประมาณ 86% และเกิด outage 5 ครั้งในวันที่ 1–2 เมษายนเพียงสองวัน GitHub ประกาศเปลี่ยนลำดับความสำคัญเป็น "Availability First" โดยจะปรับปรุง caching แยก services สำคัญออกจากกัน และย้ายส่วนที่ sensitive ด้านประสิทธิภาพไปยังระบบที่รองรับได้ดีกว่า เหตุการณ์นี้เป็นตัวอย่างที่ชัดเจนว่า AI agents กำลังเปลี่ยนแปลงโครงสร้างพื้นฐานดิจิทัลที่นักพัฒนาพึ่งพาในชีวิตประจำวันอย่างลึกซึ้งและรวดเร็วกว่าที่ใครคาดไว้

## 4. สหรัฐฯ เตรียมออกกฎหมายคุ้มครองเด็กจาก AI Chatbot กำหนดเป้าหมายที่ OpenAI และ Meta

วุฒิสภาสหรัฐฯ สนับสนุนร่างกฎหมายคุ้มครองความปลอดภัยเด็กจาก AI เมื่อวันที่ 30 เมษายน โดยมีเป้าหมายโดยตรงที่ OpenAI และ Meta ร่างกฎหมายนี้มุ่งควบคุมการเข้าถึง AI chatbot ของเด็กและเยาวชน รวมถึงกำหนดมาตรฐานการปกป้องข้อมูลและการป้องกันเนื้อหาที่เป็นอันตราย ซึ่งสะท้อนความกังวลที่เพิ่มขึ้นทั่วโลกเกี่ยวกับผลกระทบของ AI ต่อเด็ก ทั้ง OpenAI และ Meta ต่างมีผู้ใช้งานอายุน้อยจำนวนมากและอาจต้องปรับเปลี่ยนวิธีให้บริการอย่างมีนัยสำคัญ หากร่างกฎหมายผ่านการพิจารณาและลงนาม จะนับเป็นกฎระเบียบ AI สำหรับเด็กฉบับแรกของสหรัฐฯ และอาจส่งแรงกระเพื่อมต่อการออกกฎหมายในประเทศอื่นทั่วโลก

## 5. OpenAI จับมือ Yubico เปิดตัว Advanced Account Security ป้องกัน ChatGPT ด้วย Hardware Security Key

OpenAI ประกาศ Advanced Account Security (AAS) เมื่อวันที่ 30 เมษายน ร่วมกับพันธมิตร Yubico เพื่อเสริมความปลอดภัยให้กับบัญชี ChatGPT ด้วย hardware security key แบบ phishing-resistant ผู้ใช้สามารถซื้อ YubiKey co-branded สองชิ้นในราคา 68 ดอลลาร์ ซึ่งถูกกว่าราคาตลาดกว่าครึ่ง ระบบนี้บังคับใช้ passkey หรือ physical security key แทนรหัสผ่านทั่วไป โดยเฉพาะนักวิจัยด้านความปลอดภัยที่เข้าร่วมโปรแกรม Trusted Access ของ OpenAI จะต้องเปิดใช้งานภายใน 1 มิถุนายน 2026 ความเคลื่อนไหวนี้สะท้อนแนวโน้มที่ AI providers รายใหญ่กำลังยกระดับความปลอดภัยของ identity ซึ่งเป็นรากฐานสำคัญสำหรับยุคที่ AI จะฝังอยู่ในทุกงานและทุกระบบขององค์กร
