# สรุปข่าว AI ประจำวัน 28/04/2026

## 1. Microsoft และ OpenAI เขียนสัญญาพันธมิตรใหม่ เปิดทาง OpenAI บริการบน Cloud ทุกเจ้า

Microsoft และ OpenAI ประกาศปรับข้อตกลงพันธมิตรครั้งสำคัญเมื่อวันที่ 27 เมษายน 2026 ซึ่งนับเป็นการเขียนสัญญาความร่วมมือใหม่ที่มีผลไปถึงปี 2032 ภายใต้ข้อตกลงใหม่ Azure ยังคงเป็น Cloud หลักและผลิตภัณฑ์ของ OpenAI จะเปิดตัวบน Azure ก่อนเป็นอันดับแรก แต่ที่สำคัญคือ OpenAI ได้รับสิทธิ์นำผลิตภัณฑ์ทั้งหมดไปให้บริการลูกค้าผ่าน Cloud ผู้ให้บริการรายอื่น ไม่ว่าจะเป็น Amazon หรือ Google ได้แล้ว นอกจากนี้ยังกำหนดเพดานการจ่ายส่วนแบ่งรายได้จาก OpenAI ให้ Microsoft และตัดข้อกำหนดเกี่ยวกับ AGI ออกจากสัญญา การเปลี่ยนแปลงนี้สะท้อนให้เห็นว่า OpenAI มีอำนาจต่อรองสูงขึ้นมากในฐานะบริษัทที่มูลค่าและรายได้เติบโตอย่างก้าวกระโดด

🔗 [CNBC](https://www.cnbc.com/2026/04/27/openai-microsoft-partnership-revenue-cap.html)

## 2. Google ทุ่มสูงสุด $40 พันล้านลงทุนใน Anthropic สัญญาณสงครามโครงสร้างพื้นฐาน AI

Google ประกาศแผนลงทุนสูงสุดถึง 40,000 ล้านดอลลาร์ใน Anthropic เมื่อวันที่ 24 เมษายน 2026 โดยจะจ่ายทันที 10,000 ล้านดอลลาร์ที่มูลค่าบริษัท 350,000 ล้านดอลลาร์ และอีก 30,000 ล้านดอลลาร์หากบรรลุเป้าหมายที่กำหนด ข้อตกลงนี้ยังรวมถึงการจัดหากำลังประมวลผลจาก TPU ของ Google หลายกิกะวัตต์ซึ่งจะทยอยพร้อมใช้ตั้งแต่ปี 2027 น่าสังเกตว่ารายได้สะสมรายปีของ Anthropic แตะ 30,000 ล้านดอลลาร์แล้ว และนักลงทุนบางรายประเมินมูลค่าบริษัทสูงถึง 800,000 ล้านดอลลาร์ ยิ่งไปกว่านั้น Anthropic ยังเตรียมตัว IPO ในช่วงปลายปี 2026 การลงทุนมหาศาลครั้งนี้เป็นหลักฐานชัดของสงครามโครงสร้างพื้นฐาน AI ที่ทวีความเข้มข้น

🔗 [CNBC](https://www.cnbc.com/2026/04/24/google-to-invest-up-to-40-billion-in-anthropic-as-search-giant-spreads-its-ai-bets.html)

## 3. OpenAI เปิดตัว GPT-5.5 โมเดลฉลาดที่สุดในประวัติศาสตร์ มุ่งสู่ "Super App" ด้าน AI

OpenAI เปิดตัว GPT-5.5 เมื่อวันที่ 23 เมษายน 2026 โดยประกาศว่าเป็นโมเดลที่ฉลาดที่สุดและใช้งานง่ายที่สุดในประวัติศาสตร์บริษัท GPT-5.5 ทำผลลัพธ์เหนือกว่าโมเดลของ Google และ Anthropic ในหลายการทดสอบ โดยเฉพาะงาน Agentic Coding การวิจัยทางวิทยาศาสตร์ การวิเคราะห์ข้อมูล และการค้นพบยา แม้ราคาจะสูงกว่า GPT-5.4 แต่ประสิทธิภาพการใช้ Token ที่ดีกว่าทำให้ต้นทุนจริงลดลง จุดที่น่าจับตาคือ Greg Brockman ประธาน OpenAI ระบุว่า GPT-5.5 นำบริษัทเข้าใกล้เป้าหมาย "Super App" ที่จะรวม ChatGPT, Codex และ AI Browser ไว้ในแพลตฟอร์มเดียว ซึ่งอาจเปลี่ยนรูปแบบการใช้งาน AI ในชีวิตประจำวันอย่างสิ้นเชิง

🔗 [TechCrunch](https://techcrunch.com/2026/04/23/openai-chatgpt-gpt-5-5-ai-model-superapp/)

## 4. Z.ai ปล่อย GLM-5.1 โอเพนซอร์สทำงานอิสระได้ 8 ชั่วโมง แซง Claude และ GPT บน SWE-Bench

Z.ai (เดิมชื่อ Zhipu AI บริษัทจดทะเบียนในฮ่องกง) เปิดตัว GLM-5.1 เมื่อวันที่ 7 เมษายน 2026 โมเดลโอเพนซอร์สขนาด 754 พันล้านพารามิเตอร์นี้ทำคะแนนสูงสุดในโลกที่ 58.4 บน SWE-Bench Pro แซงหน้า GPT-5.4 (57.7) และ Claude Opus 4.6 (57.3) ความสามารถที่โดดเด่นที่สุดคือการทำงานแบบ Agentic ได้นานถึง 8 ชั่วโมงโดยไม่ต้องมีมนุษย์แทรกแซง ครอบคลุมวงจรตั้งแต่ "วางแผน → ดำเนินการ → ทดสอบ → แก้ไข → ปรับปรุง" ทีมสาธิตโดยให้ GLM-5.1 สร้าง Linux Desktop Environment ตั้งแต่ศูนย์ผ่าน 655 รอบซ้ำ โมเดลเผยแพร่ภายใต้ MIT License เปิดกว้างสำหรับทุกการใช้งาน ถือเป็นก้าวสำคัญของ AI โอเพนซอร์สจากเอเชียที่สามารถแข่งขันกับระบบปิดของตะวันตกได้แล้ว

🔗 [VentureBeat](https://venturebeat.com/technology/ai-joins-the-8-hour-work-day-as-glm-ships-5-1-open-source-llm-beating-opus-4)

## 5. Google Cloud Next 2026 เปิดตัวแพลตฟอร์ม AI Agent และชิป TPU รุ่น 8 แข่ง Nvidia

Google จัดงาน Cloud Next 2026 เมื่อวันที่ 22 เมษายน 2026 โดยมีประกาศสำคัญสองด้านหลัก ด้านแรกคือการเปิดตัว Gemini Enterprise Agent Platform ระบบครบวงจรสำหรับสร้าง จัดการ และกำกับดูแล AI Agent ในองค์กร ประกอบด้วย Agent Studio, Agent Registry, Agent Gateway และ Agent Observability ด้านที่สองคือการเปิดตัว TPU รุ่นที่ 8 สองแบบ ได้แก่ TPU 8t สำหรับการเทรนที่รองรับ 9,600 TPU พร้อมหน่วยความจำ 2 เพตาไบต์ และ TPU 8i สำหรับ Inference ที่รองรับ AI Agent หลายล้านตัวพร้อมกัน Google ยังเปิดตัว Virgo Network เชื่อมต่อ Supercomputer ขนาดมหึมา การประกาศเหล่านี้แสดงว่า Google กำลังเดิมพันอย่างจริงจังกับ "Agentic Era" อันเป็นทิศทางหลักของ AI ในปี 2026

🔗 [Google Blog](https://blog.google/innovation-and-ai/infrastructure-and-cloud/google-cloud/google-cloud-next-26-recap/)
