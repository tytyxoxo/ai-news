## 1. Microsoft และ OpenAI ปรับโครงสร้างพันธมิตร เปิดทาง OpenAI ให้บริการข้าม Cloud ได้

Microsoft และ OpenAI ประกาศปรับปรุงข้อตกลงพันธมิตรครั้งสำคัญเมื่อวันที่ 27 เมษายน โดย Microsoft ยังคงเป็นพาร์ทเนอร์ Cloud หลักที่ผลิตภัณฑ์ OpenAI จะเปิดตัวบน Azure ก่อน แต่ OpenAI ได้รับสิทธิ์ให้บริการลูกค้าผ่าน Cloud รายอื่นได้ด้วย ซึ่งเป็นการเปิดกว้างเชิงกลยุทธ์ที่สำคัญและเพิ่มความยืดหยุ่นให้ทั้งสองฝ่ายอย่างมีนัยสำคัญ

ที่มา: Microsoft Blog

## 2. OpenAI เปิดตัว GPT-5.5 โมเดลทรงพลังที่สุดสำหรับงาน Coding อัตโนมัติ

OpenAI เปิดตัว GPT-5.5 เมื่อวันที่ 23 เมษายน ประกาศว่าเป็นโมเดลที่ฉลาดและใช้งานง่ายที่สุดที่บริษัทเคยสร้างมา โมเดลนี้ทำคะแนน 82.7% บน Terminal-Bench 2.0 และ 58.6% บน SWE-Bench Pro กลายเป็นโมเดล Agentic Coding ที่แข็งแกร่งที่สุดในตลาด ณ ขณะนี้ และสามารถทำงานจบครบในรอบเดียวได้มากกว่ารุ่นก่อนหน้า

ที่มา: TechCrunch

## 3. Google ประกาศลงทุนสูงสุด 4 หมื่นล้านดอลลาร์ใน Anthropic

Google ประกาศแผนลงทุนสูงสุดถึง 40,000 ล้านดอลลาร์ใน Anthropic บริษัท AI ผู้สร้าง Claude โดยการสนับสนุนครอบคลุมทั้งเงินทุนและทรัพยากรการประมวลผล การลงทุนขนาดมหาศาลนี้สะท้อนให้เห็นถึงความตั้งใจของ Google ในการรักษาความได้เปรียบในการแข่งขัน AI ระดับ Frontier และเสริมความแข็งแกร่งให้กับระบบนิเวศ AI ในระยะยาว

ที่มา: TechCrunch

## 4. DeepSeek เปิดตัว V4 ประสิทธิภาพระดับ Frontier ราคาถูกกว่าคู่แข่งมาก

สตาร์ทอัพ AI จากจีน DeepSeek เปิดตัวโมเดลใหม่ DeepSeek V4 เวอร์ชันพรีวิวสองรุ่นเมื่อวันที่ 24 เมษายน โมเดลนี้ถูกออกแบบมาเพื่อ "ปิดช่องว่าง" กับโมเดลระดับ Frontier จากฝั่งตะวันตก ขณะที่มีราคาใช้งานถูกกว่าโมเดลชั้นนำในตลาดปัจจุบันอย่างมีนัยสำคัญ ทำให้เป็นตัวเลือกที่น่าสนใจสำหรับนักพัฒนาและองค์กรที่คำนึงถึงต้นทุน

ที่มา: TechCrunch

## 5. Meta เปิดตัว Muse Spark โมเดล AI เชิงพาณิชย์ตัวแรกหลังก่อตั้ง Superintelligence Labs

Meta เปิดตัว Muse Spark โมเดล AI เชิงพาณิชย์ตัวแรกนับตั้งแต่จัดตั้ง Superintelligence Labs โดยโมเดลใช้เทคนิค "thought compression" ที่ช่วยให้ทำงานเหตุผลเชิงซับซ้อนด้วยพลังประมวลผลน้อยกว่า Llama 4 Maverick มากกว่า 10 เท่า ถือเป็นก้าวสำคัญที่นำ Meta กลับเข้าสู่กลุ่ม 5 โมเดลชั้นนำระดับโลกอีกครั้ง
