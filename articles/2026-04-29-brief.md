## 1. Google ทุ่ม 4 หมื่นล้านดอลลาร์ลงทุนใน Anthropic สร้างพันธมิตรยักษ์ใหญ่ด้าน AI

Google ประกาศแผนการลงทุนสูงสุดถึง 40,000 ล้านดอลลาร์สหรัฐใน Anthropic บริษัทสตาร์ทอัป AI ที่ก่อตั้งโดยอดีตทีมงานจาก OpenAI โดยนอกจากเงินลงทุนแล้ว Google ยังให้การสนับสนุนด้านโครงสร้างพื้นฐานการประมวลผลแก่ Anthropic ด้วย ดีลนี้เกิดขึ้นในช่วงที่ Amazon ก็ได้ลงทุนเพิ่มอีก 5,000 ล้านดอลลาร์ใน Anthropic เช่นกัน ซึ่งสะท้อนให้เห็นถึงการแข่งขันอย่างดุเดือดระหว่างบริษัทเทคโนโลยียักษ์ใหญ่ในการครองตลาด AI รุ่นใหม่ การลงทุนครั้งนี้แสดงให้เห็นว่า Anthropic กลายเป็นตัวแสดงสำคัญในสมรภูมิ AI ที่ทั้ง Google, Amazon และ Microsoft ต่างงัดทรัพยากรมหาศาลออกมาแย่งชิงพันธมิตรเชิงยุทธศาสตร์

## 2. Microsoft ประกาศก้าวใหม่ของพันธมิตรกับ OpenAI เปิดประตูให้ลูกค้าทุก Cloud

Microsoft ได้ออกประกาศเกี่ยวกับทิศทางใหม่ของความร่วมมือกับ OpenAI เมื่อวันที่ 27 เมษายน 2026 โดยข้อตกลงใหม่กำหนดให้ Microsoft ยังคงเป็นพาร์ตเนอร์ Cloud หลักของ OpenAI และผลิตภัณฑ์ OpenAI จะเปิดตัวบน Azure ก่อนเป็นอันดับแรก อย่างไรก็ตาม OpenAI ได้รับอิสระในการให้บริการผลิตภัณฑ์แก่ลูกค้าบน Cloud ผู้ให้บริการรายอื่นได้ด้วย ซึ่งหมายความว่า Amazon Web Services ก็สามารถนำเสนอโมเดลล่าสุดของ OpenAI รวมถึง Codex และเครื่องมือสร้าง AI agent บนแพลตฟอร์ม Bedrock ได้แล้ว ความเปลี่ยนแปลงนี้สะท้อนถึงพลวัตที่เปลี่ยนไปในอุตสาหกรรม AI ที่ OpenAI เริ่มขยายอำนาจต่อรองและช่องทางจัดจำหน่ายของตัวเองมากขึ้น

## 3. DeepSeek เปิดตัว V4 โมเดล AI ที่ถูกกว่าคู่แข่งระดับ Frontier อย่างเห็นได้ชัด

ค่าย DeepSeek จากจีนได้เปิดตัวโมเดลภาษาขนาดใหญ่รุ่นใหม่ชื่อ DeepSeek V4 ซึ่งเป็นการอัปเดตจาก V3.2 ที่เปิดตัวปีที่แล้ว โดยมาในสองเวอร์ชัน ได้แก่ รุ่นเต็มและรุ่น V4 Flash ที่เบากว่า จุดเด่นที่สำคัญที่สุดของ DeepSeek V4 คือราคาที่ถูกกว่าโมเดลระดับ Frontier ทั่วไปอย่างมาก โดย V4 Flash มีราคาเพียง 0.14 ดอลลาร์ต่อล้าน input token ซึ่งต่ำกว่าคู่แข่งสำคัญหลายเท่า DeepSeek ยืนยันว่าโมเดลนี้สามารถ "ปิดช่องว่าง" กับโมเดลชั้นนำของโลกได้ การเปิดตัวครั้งนี้ยิ่งเพิ่มแรงกดดันด้านราคาให้กับผู้เล่นรายอื่นในตลาด AI ทั่วโลก และอาจเร่งให้เกิดการแข่งขันด้านต้นทุนในอุตสาหกรรมอย่างรุนแรงยิ่งขึ้น

## 4. OpenAI อาจพัฒนาสมาร์ทโฟนแนวคิดใหม่ที่ AI agent ทำหน้าที่แทนแอปทั่วไป

นักวิเคราะห์ชื่อดัง Ming-Chi Kuo รายงานว่า OpenAI อาจกำลังพัฒนาสมาร์ทโฟนรุ่นใหม่ร่วมกับพันธมิตรอย่าง MediaTek, Qualcomm และ Luxshare โดยแนวคิดของโทรศัพท์เครื่องนี้แตกต่างจากสมาร์ทโฟนทั่วไปอย่างสิ้นเชิง เนื่องจากผู้ใช้จะพึ่งพา AI agent ในการทำงานต่างๆ แทนที่จะเปิดแอปพลิเคชันแยกกันอย่างที่เคยทำ หากแนวคิดนี้เป็นจริง อาจหมายถึงการเปลี่ยนแปลงครั้งใหญ่ในรูปแบบการใช้งานโทรศัพท์มือถือ และเป็นความท้าทายครั้งสำคัญต่อ Apple และ Google ที่ครองตลาดระบบปฏิบัติการมือถือมาอย่างยาวนาน

## 5. Z.ai เปิดตัว GLM-5.1 โอเพนซอร์ส 754 พันล้านพารามิเตอร์ ทำงานอิสระได้ 8 ชั่วโมง

Z.ai บริษัทเทคโนโลยีจีนที่เพิ่งเข้าตลาดหลักทรัพย์ฮ่องกงด้วยมูลค่า 52,830 ล้านดอลลาร์ ได้เปิดตัวโมเดลภาษา GLM-5.1 ที่มีขนาดถึง 754,000 ล้านพารามิเตอร์ พร้อม context window ยาว 202,752 token ในการทดสอบสำคัญที่ให้โมเดลทำงาน 8 ชั่วโมงโดยไม่มีคนควบคุม GLM-5.1 สามารถสร้างสภาพแวดล้อม Desktop แบบ Linux ได้ตั้งแต่ต้น รวมถึง file browser, terminal, text editor และเกมที่เล่นได้จริง ที่น่าสนใจคือโมเดลนี้ทำคะแนน SWE-Bench Pro สูงกว่า Claude Opus 4.6 และ GPT-5.4 แสดงให้เห็นว่าโมเดลโอเพนซอร์สจากจีนกำลังก้าวขึ้นมาท้าทายโมเดลชั้นนำจากสหรัฐฯ อย่างจริงจัง
