## 1. Google ลงนามข้อตกลงลับกับกระทรวงกลาโหมสหรัฐฯ เปิดให้กองทัพใช้ Gemini AI ในงานลับ แม้พนักงานนับร้อยประท้วง

เมื่อวันที่ 28 เมษายน 2026 Google ลงนามข้อตกลงลับกับกระทรวงกลาโหมสหรัฐฯ (Pentagon) อนุญาตให้กองทัพนำโมเดล AI ขั้นสูงของบริษัท รวมถึง Gemini ไปใช้ใน "งานรัฐบาลที่ชอบด้วยกฎหมายทุกประเภท" ซึ่งรวมถึงภารกิจลับ ก่อนการประกาศเพียงไม่กี่ชั่วโมง พนักงานกว่า 600 คนรวมถึงผู้บริหารระดับสูงจาก DeepMind ได้ลงชื่อในจดหมายเปิดผนึกขอให้ CEO Sundar Pichai ปฏิเสธข้อตกลงนี้ด้วยความกังวลว่า AI อาจถูกนำไปใช้ใน "วิธีที่โหดร้ายหรือเป็นอันตรายอย่างยิ่ง" หลังจากเข้าสู่โลกของปฏิบัติการลับที่ขาดความโปร่งใส ข้อตกลงนี้มาในช่วงเดียวกับที่ Anthropic ปฏิเสธข้อเสนอลักษณะเดียวกัน ทำให้ Google กลายเป็นผู้จัดหา AI ลับรายล่าสุดต่อจาก OpenAI และ xAI และจุดชนวนการถกเถียงใหม่เกี่ยวกับจริยธรรม AI ในบริบทการทหาร

## 2. OpenAI ยุติสัญญาผูกขาดกับ Microsoft โมเดล OpenAI พร้อมใช้งานบน Amazon AWS แล้ววันนี้

เมื่อวันที่ 28 เมษายน 2026 เพียงหนึ่งวันหลัง OpenAI และ Microsoft ประกาศยกเลิกข้อตกลงผูกขาดที่ผูกพันมาตั้งแต่ปี 2019 Amazon Web Services ก็เปิดตัวโมเดล OpenAI บน Amazon Bedrock ทันที ครอบคลุม GPT รุ่นล่าสุด Codex สำหรับงานเขียนโค้ด และ Amazon Bedrock Managed Agents ในโหมด limited preview ข้อตกลงใหม่ให้ Microsoft มีสิทธิ์ใช้ IP ของ OpenAI แบบไม่ผูกขาดถึงปี 2032 แทนที่สัญญาเดิมที่ให้สิทธิ์ Azure แต่เพียงผู้เดียว ตัดทิ้งข้อกำหนดที่จะเปลี่ยนความสัมพันธ์เมื่อบรรลุ AGI ด้วย การปลดล็อกนี้เป็นผลพวงโดยตรงจากดีล $50,000 ล้านดอลลาร์กับ Amazon ที่ประกาศในเดือนกุมภาพันธ์ และถือเป็นจุดเปลี่ยนสำคัญของตลาด Cloud AI ที่ต่อไป AWS, Azure และ Google Cloud จะแข่งขันกันให้บริการโมเดล OpenAI พร้อมกัน

## 3. งานวิจัย ICLR 2026 เผย "กับดักเหตุผล" — ยิ่งฝึก AI ให้คิดเก่ง ยิ่งหลอนเรียกใช้เครื่องมือที่ไม่มีจริงมากขึ้น

งานวิจัย "The Reasoning Trap: How Enhancing LLM Reasoning Amplifies Tool Hallucination" ที่นำเสนอในงาน ICLR 2026 ที่เมืองริโอ เผยผลที่น่ากังวลสำหรับทีมพัฒนา AI Agent: การฝึกโมเดลด้วย Reinforcement Learning เพื่อเพิ่มความสามารถด้านเหตุผล มีความสัมพันธ์เชิงสาเหตุโดยตรงกับการเพิ่มขึ้นของ tool hallucination ซึ่งคือพฤติกรรมที่โมเดลอ้างถึงหรือพยายามเรียกใช้เครื่องมือที่ไม่มีอยู่จริงในระบบ ทีมวิจัยพัฒนา benchmark ใหม่ชื่อ SimpleToolHalluBench เพื่อวัดปัญหานี้ใน 2 สถานการณ์ ได้แก่ กรณีไม่มีเครื่องมือที่ตรงกับงาน และกรณีมีเครื่องมือล่อที่ดูเหมือนเกี่ยวข้อง ข้อค้นพบนี้มีนัยสำคัญในทางปฏิบัติ คือองค์กรที่กำลังนำ AI Agent ที่ใช้ reasoning model เข้าสู่สภาพแวดล้อมการผลิตจริง จำเป็นต้องออกแบบระบบควบคุมเครื่องมืออย่างรัดกุมมากกว่าเดิม

ที่มา: OpenReview (ICLR 2026)

## 4. OpenAI เปิดโปรแกรม Bio Bug Bounty สำหรับ GPT-5.5 รางวัลสูงสุด 25,000 ดอลลาร์ สำหรับนักวิจัยที่ค้นพบช่องโหว่ด้านชีวภาพ

เมื่อวันที่ 28 เมษายน 2026 OpenAI เปิดตัวโปรแกรม "GPT-5.5 Bio Bug Bounty" การทดสอบความปลอดภัยแบบ red-teaming ที่มุ่งเน้นความเสี่ยงด้านชีววิทยา (biosafety) โดยเฉพาะ นักวิจัยที่ผ่านการตรวจสอบประวัติต้องพยายามหา "universal jailbreak" ที่สามารถหลบเลี่ยงระบบกรองเนื้อหาของ GPT-5.5 ในชุดคำถามด้านชีวภาพที่กำหนดไว้ 5 ข้อ รางวัลสูงสุดอยู่ที่ 25,000 ดอลลาร์ โปรแกรมทดสอบจริงตั้งแต่ 28 เมษายน ถึง 27 กรกฎาคม 2026 และไม่เปิดสาธารณะ ผู้เข้าร่วมต้องลงนาม NDA การเปิดโปรแกรมนี้สะท้อนถึงความกังวลที่เพิ่มขึ้นในวงการ AI ว่าโมเดลภาษาขั้นสูงอาจถูกใช้เป็นเครื่องมือช่วยออกแบบสารชีวภาพอันตราย และแสดงให้เห็นว่า OpenAI กำลังลงทุนกับกระบวนการความปลอดภัยเชิงรุกมากขึ้น

## 5. YouTube ทดสอบฟีเจอร์ AI Search แบบใหม่ ตอบคำถามซับซ้อนด้วยการมิกซ์วิดีโอและข้อความแบบ Step-by-Step

เมื่อวันที่ 28 เมษายน 2026 YouTube เปิดทดสอบฟีเจอร์ "Ask YouTube" ที่ให้ผู้ใช้ถามคำถามซับซ้อนในแถบค้นหาได้โดยตรง เช่น "วางแผนทริป 3 วันจากซานฟรานซิสโกถึงซานตาบาร์บาร่า" และรับคำตอบในรูปแบบผสมระหว่างข้อความอธิบาย คลิปสั้น และวิดีโอยาวในลักษณะ step-by-step พร้อมถามต่อยอดได้ เช่น "ที่ไหนมีกาแฟดี?" ฟีเจอร์นี้เป็นส่วนหนึ่งของการขยาย AI Mode ของ Google ไปยังทุก surface หลังจากที่เปิดตัวในการค้นหาหลักแล้ว แม้ดูเหมือนเป็นฟีเจอร์สำหรับผู้บริโภค แต่ผลกระทบระยะยาวต่อ creator และ SEO มีนัยสำคัญ เมื่อผู้ใช้ได้รับคำตอบสังเคราะห์จาก AI โดยตรงแทนการคลิกเข้าดูวิดีโอ ซึ่งอาจเปลี่ยนพฤติกรรมการบริโภคเนื้อหาวิดีโอในระยะยาว
