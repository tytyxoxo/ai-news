## 1. Microsoft ปรับโครงสร้างความร่วมมือกับ OpenAI ครั้งใหญ่ เปิดทางให้ใช้ AWS และ Google Cloud ได้

เมื่อวันที่ 27 เมษายน 2026 Microsoft และ OpenAI ประกาศปรับโครงสร้างความร่วมมือที่ผูกมัดกันมานานอย่างรุนแรงที่สุดนับตั้งแต่เริ่มต้น โดยยกเลิกข้อผูกมัดด้านคลาวด์แบบ exclusive ทำให้ OpenAI สามารถให้บริการผ่าน AWS, Google Cloud หรือผู้ให้บริการอื่นได้โดยไม่ต้องจำกัดอยู่กับ Azure อีกต่อไป แม้ Azure ยังคงเป็น "primary partner" ที่ได้รับสิทธิ์ launch ผลิตภัณฑ์ก่อน Microsoft ยังคงใบอนุญาต IP ของ OpenAI ถึงปี 2032 และถือหุ้นใหญ่ต่อไป ขณะที่ OpenAI ยังแบ่งรายได้ 20% ให้ Microsoft ถึงปี 2030 แต่มีเพดานชัดเจน การเปลี่ยนแปลงนี้สะท้อนว่าทั้งสองฝ่ายต้องการอิสระมากขึ้นเพื่อรับมือการแข่งขัน AI ที่เดือดระอุ

## 2. Microsoft เปิดตัว 3 โมเดล MAI ใหม่ ท้าชน OpenAI Whisper และ Google ในตลาด AI foundation model

ต้นเดือนเมษายน 2026 Microsoft เปิดตัวโมเดล AI ขั้นพื้นฐาน 3 รายการที่พัฒนาเองทั้งหมด ได้แก่ MAI-Transcribe-1 ระบบถอดความเสียงที่อ้างว่าแม่นยำกว่า Whisper ของ OpenAI ใน 25 ภาษา, MAI-Voice-1 ที่สร้างเสียงธรรมชาติ 60 วินาทีภายในเพียง 1 วินาทีและรองรับการโคลนเสียงจากตัวอย่างสั้นๆ และ MAI-Image-2 โมเดลสร้างภาพที่ติดท็อป 3 บน Arena.ai leaderboard พร้อมความเร็วเร็วขึ้น 2 เท่า โมเดลทั้งสามพร้อมให้ใช้งานผ่าน Microsoft Foundry และ MAI Playground แล้ว ถือเป็นก้าวสำคัญที่แสดงว่า Microsoft กำลังสร้างศักยภาพ AI ของตัวเองโดยไม่พึ่งพา OpenAI เพียงอย่างเดียว

## 3. Z.ai เปิดตัว GLM-5.1 โมเดล open source ที่ทำงานอัตโนมัติได้ 8 ชั่วโมง สู้ ChatGPT บน SWE-Bench

สตาร์ทอัพ AI จากจีน Z.ai เปิดตัว GLM-5.1 ภายใต้ใบอนุญาต MIT ให้ใช้งานเชิงพาณิชย์ได้อย่างเสรี โมเดลนี้มีขนาด 754 พันล้านพารามิเตอร์แบบ Mixture-of-Experts ออกแบบมาให้ทำงานอิสระได้นานถึง 8 ชั่วโมงต่อเนื่องในงาน coding ซับซ้อน ซึ่งเป็นนวัตกรรมที่ยังไม่มีในโมเดล open source ตัวไหนก่อนหน้า GLM-5.1 ครองอันดับ 1 ของโมเดล open source บน SWE-Bench Pro ด้วยคะแนน 58.4 เหนือทั้ง Claude Opus 4.6 และ GPT-5.4 ในงาน coding ระยะยาว ราคา API อยู่ที่เพียง $1.40 ต่อล้าน input token การเปิดตัวครั้งนี้ส่งสัญญาณว่าสงครามโมเดล AI ไม่ได้จำกัดแค่ฝั่งตะวันตกอีกต่อไป

## 4. Google ทำสัญญาหลายพันล้านกับ Thinking Machines Lab ของ Mira Murati อดีต CTO OpenAI

เมื่อวันที่ 22 เมษายน 2026 TechCrunch รายงานว่า Google Cloud ทำข้อตกลงมูลค่าหลายพันล้านดอลลาร์กับ Thinking Machines Lab (TML) สตาร์ทอัพ AI ที่ก่อตั้งโดย Mira Murati อดีต CTO ของ OpenAI ข้อตกลงนี้ให้ TML เข้าถึงชิป Nvidia GB300 รุ่นล่าสุดผ่าน Google Cloud พร้อมโครงสร้างพื้นฐานสำหรับฝึกโมเดลและ deployment รองรับ reinforcement learning workloads ซึ่งเป็นหัวใจของ "Tinker" เครื่องมือสร้าง frontier AI model อัตโนมัติของ TML การลงทุนของ Google ใน TML เสริมกับที่ลงทุนใน Anthropic กว่า 40,000 ล้านดอลลาร์ สะท้อนกลยุทธ์ Google ที่ต้องการเป็น cloud provider หลักของสตาร์ทอัพ AI ชั้นนำทุกราย

## 5. Meta ทำสัญญา Amazon ใช้ชิป CPU แทน GPU สำหรับ AI agent บอกรับยุคใหม่ของ AI infrastructure

เมื่อวันที่ 24 เมษายน 2026 Amazon ประกาศว่า Meta ทำสัญญาใช้ชิป AWS Graviton ซึ่งเป็น ARM-based CPU จำนวนหลายล้านตัวสำหรับรองรับ workload ของ AI agent โดยเฉพาะ นี่เป็นสัญญาณสำคัญว่าอุตสาหกรรม AI กำลังก้าวเข้าสู่ยุคใหม่ที่ CPU กลายมามีบทบาทสำคัญไม่แพ้ GPU เมื่อโมเดลใหญ่ฝึกเสร็จแล้ว งาน AI agent เช่น real-time reasoning การเขียนโค้ด การค้นหา และการประสานงาน multi-step กลับเหมาะกับ CPU มากกว่า Graviton รุ่นล่าสุดของ Amazon ออกแบบมาสำหรับ AI โดยเฉพาะ ดีลครั้งนี้เป็นหลักฐานสำคัญที่ Amazon ใช้โชว์ว่า CPU ตัวเองสามารถแข่งขันได้จริงในตลาด AI infrastructure
